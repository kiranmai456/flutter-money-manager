# 💰 Flutter Money Manager

A modern personal finance application built with **Flutter** and **Dart** that helps users manage their income and expenses efficiently. The application provides transaction tracking, financial summaries, and a clean, responsive user interface while following a modular project architecture.

---

## ✨ Features

* 💵 Add Income
* 💸 Add Expenses
* 📋 View Transaction History
* 📊 Financial Summary
* 🎨 Custom Application Theme
* 📱 Responsive UI
* 🗂️ Modular Project Architecture
* ⚡ Smooth Navigation

---

## 🛠️ Tech Stack

* 💙 Flutter
* 🎯 Dart
* 🎨 Material Design
* 📱 Responsive UI
* 🧩 Modular Architecture

---

## 📂 Project Structure

```text
lib/
│
├── controllers/
├── pages/
├── assets/
├── theme.dart
├── static.dart
└── main.dart
```

### Folder Description

* **controllers/** – Contains business logic for transaction management.
* **pages/** – Application screens such as Dashboard, Income, Expense, and Add Transaction.
* **assets/** – Images, icons, and static resources.
* **theme.dart** – Centralized colors, typography, and application themes.
* **static.dart** – Shared constants and reusable values.
* **main.dart** – Entry point of the application.

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/kiranmai456/flutter-money-manager.git
```

### Navigate to the project

```bash
cd flutter-money-manager
```

### Install dependencies

```bash
flutter pub get
```

### Run the application

```bash
flutter run
```
